Данная программа заполняет учет рабочего времени на портале AMT.
Если вам необходимо заполнить дни определенной недели, то вам следует указать диапазон дат этой недели, следуя инструкциям, которые выдаст программа.
Если какие-то дни в заданном вами промежутку дат уже заполнены, то программа это увидит и пропустит данные дни.
Необходимые зависимости описаны в файле "requirements.txt"

v1.0 - особенности работы
- Рабочее время всегда прописывается с 10 утра до 19 вечера
- Значение для поля "название работы" берется рандомно из файла "nazvanie_rabot.txt"
- Выбор места работы (в офисе, вне офиса, дома...) выбирается сразу для всех дат в заданном диапазоне.
- Поле "Проект" не заполняется, так как является необязательным.
